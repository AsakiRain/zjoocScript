 <h1>使用教程</h1>
 1.下油猴<br>
 2.下脚本<br>
 3.把脚本装到油猴<br>
 4.打开网课视频列表，从左侧章节栏选择想要开始的视频，刷新，<b>重新从左侧章节栏选择视频</b><br>
